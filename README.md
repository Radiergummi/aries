# aries

A web framework for C++ language.

## Build
```
git clone https://github.com/itpkg/aries.git
cd aries

mkdir build
cd build
cmake -DCMAKE_BUILD_TYPE=Release -DCMAKE_C_COMPILER=clang -DCMAKE_CXX_COMPILER=clang++ ..
make
make install

./demo
```
